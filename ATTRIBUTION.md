# Attribution

## Why Attribute?

Triality is open-source software released under the MIT License. Attribution is
not legally required — but it matters.

Every mention of Triality helps researchers, engineers, and organizations
discover a new class of tooling: computational systems designed to reason about
the physical world. Attribution is how open infrastructure grows. When you
reference Triality, you are contributing to a shared effort to make physics-aware
computation accessible and widely adopted.

---

## How to Attribute

Choose whichever format fits your context. The goal is visibility, not
rigidity — adapt these to your medium.

### In-Product or Application UI

```
Powered by Triality
```

```
Powered by Triality (Mentis OS) — genovationsolutions.com
```

### In Documentation or Technical Writing

> This system uses [Triality](https://genovationsolutions.com), a physics
> reasoning engine developed by Genovation Technological Solutions Pvt Ltd.

### In Source Code (comments or config)

```
# Physics reasoning provided by Triality (https://genovationsolutions.com)
# Copyright (c) 2024-2026 Genovation Technological Solutions Pvt Ltd.
# Licensed under the MIT License.
```

### Badge (Markdown)

```markdown
[![Powered by Triality](https://img.shields.io/badge/Powered%20by-Triality-blue)](https://genovationsolutions.com)
```

### Badge (HTML)

```html
<a href="https://genovationsolutions.com">
  <img src="https://img.shields.io/badge/Powered%20by-Triality-blue" alt="Powered by Triality">
</a>
```

## Academic Citation

If Triality contributes to your research, please cite it formally. This ensures
proper credit in the academic record and helps others trace the tools used in
published work.

**Inline citation:**

> Genovation Technological Solutions Pvt Ltd. (2024). *Triality: A Computational
> Substrate for Physically Grounded Reasoning.*

**BibTeX:**

```bibtex
@software{triality2024,
  title     = {Triality: A Computational Substrate for Physically Grounded Reasoning},
  author    = {{Genovation Technological Solutions Pvt Ltd.}},
  year      = {2024},
  url       = {https://genovationsolutions.com}
}
```
